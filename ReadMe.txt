To properly execute this code, you will need:

MySQL Workbench (current version)
MySQL for Visual Studio (current version)
MySQL Connector for ADO.NET (current version)

The database will need the following attributes
"Server = localhost; Database=libdb;Uid = root;password=root"
This is the connection string for DBHelper(). 

The "Populate DB" button will load some basic tables to test functionalities.
